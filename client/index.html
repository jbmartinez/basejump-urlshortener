API Basejump: URL Shortener microservice

What does this do?

  • Pass a URL as a parameter and you will receive a shortened URL in the JSON response.
  • If yiu pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When you visit that shortened URL, it will redirect you to the original link.

Example usage

Requests

To create a new shortcut use:

https://hidden-cove-11405.herokuapp.com/new/http://freecodecamp.com/news

Which generates a response similar to this 

{"shortcut":"http://hidden-cove-11405.herokuapp.com/0S016Oa","url":"http://freecodecamp.com/news"}

Visiting http://hidden-cove-11405.herokuapp.com/0S016Oa will redirect you to http://freecodecamp.com/news
